### Assignment Workflow

- **Assignment settings**: Added instructions, answer reveal, item shuffle,
  attempt limits, timers, expiry windows, and close or reopen controls.
- **Stable snapshots**: Published assignments freeze the student-facing content
  so later activity edits do not rewrite already-shared homework links.
- **Lifecycle helpers**: Shared open, closed, and expired-state logic across
  public student access, teacher lists, and results pages.
- **Safe public runner**: Student links receive sanitized runtime items and keep
  correct answers hidden until scoring and allowed review.
- **Rule summaries**: Public student runners now frame item count, attempts,
  timer, close time, identity mode, and review behavior before submission.

### Results and Exports

- **Teacher summaries**: Added completions, average accuracy, average duration,
  student summaries, reteach priorities, and full item-performance tables.
- **Answer review**: Preserved expected answers, accepted alternatives,
  explanations, and per-item student responses from the assignment snapshot.
- **CSV export**: Added private results export for gradebooks, parent follow-up,
  offline records, and classroom review.
- **Copy-ready review artifacts**: Classroom briefs, reteach plans, item review,
  and student follow-up summaries now preserve snapshot-based result context
  without exposing raw anonymous browser tokens.

### AI and Template Readiness

- **AI draft boundary**: AI-assisted creation returns reviewable activity drafts
  instead of publishing directly.
- **Source-material safety**: AI draft summaries distinguish safe source
  provenance from omitted private file metadata before extraction tooling is
  designed.
- **Template readiness**: Activities can show which templates are immediately
  playable and which need more structured content.
- **Deterministic quiz choices**: Sparse quiz options can be completed from
  sibling answers and vocabulary while preserving the teacher's correct answer.
- **Public content cleanup**: Homepage, legal pages, blog posts, and release
  notes now describe ClassGamify rather than the copied starter product.
