### New Product Surfaces

- **Worksheet generator**: Added paper-size aware worksheet previews, print
  settings, assignment notes, custom character lists, recent worksheet sets,
  and clean print output.
- **Assignment handoff**: Added copyable worksheet links and assignment messages
  so teachers, parents, tutors, and learners can continue the same set later.
- **Teacher and parent page**: Added a focused workflow page for classroom,
  tutoring, and family practice.
- **Pricing guidance**: Clarified the difference between Free, HSK1 Pro, Early
  Lifetime, and classroom inquiry paths.

### Trust and Commercial Readiness

- Improved privacy policy, terms, cookie policy, and account copy so the site
  reads like a live product rather than a generic starter.
- Added clearer billing language around free access, paid plan value, hosted
  checkout, and lifetime access.
- Added structured classroom inquiry fields so high-intent users can explain
  learner count, level, weekly routine, and worksheet needs.

### UX Quality

- Tightened footer, navigation, dashboard, practice, worksheet, and pricing
  flows around the same daily learning loop.
- Added clearer local-progress backup guidance for learners who may switch
  devices or hand off progress.
- Continued responsive cleanup to reduce horizontal overflow and keep dense
  controls usable on mobile and desktop.
