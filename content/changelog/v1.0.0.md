### Learning Experience

This release turned Lang Study into a usable Chinese character practice product:

- **HSK1 starter path**: A focused set of beginner characters with pinyin,
  meaning, examples, and study cues.
- **Stroke-order practice**: Hanzi Writer powered animation and tracing so
  learners can watch, try, and repeat each character.
- **Local progress**: Browser-saved completion state, missed strokes, review
  status, and daily rhythm signals.
- **Printable worksheets**: A first worksheet path that moves the same
  character set from screen practice to paper handwriting.

### Product Direction

- Positioned Lang Study around repeatable writing practice rather than generic
  language content.
- Added bilingual English and Chinese surfaces for the core learning flow.
- Started shaping teacher, parent, tutor, and self-study use cases around the
  same practice loop.

### Quality

- Kept the initial scope narrow so the public product is immediately useful:
  practice one set, review missed strokes, then print.
- Added SEO metadata and structured learning-page context for the main HSK1
  surfaces.
