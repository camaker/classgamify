### Classroom Foundation

This release moved the copied starter project toward the ClassGamify classroom
loop:

- **Activity model**: Introduced reusable teacher-owned activities with
  template-neutral lesson content.
- **Template catalog**: Added first support for quiz, match-up, line-match,
  group sort, fill-blank, listening, matching pairs, and open-box workflows.
- **Student runner**: Started the public assignment-play surface with
  template-aware interactions and shared answer submission shape.
- **Teacher dashboard**: Added first activity library and assignment list
  surfaces so teachers can create, reopen, and publish classroom work.

### Product Direction

- Positioned ClassGamify around the Wordwall-style loop of create, publish,
  play, and review.
- Kept templates tied to one structured content model so future games do not
  create disconnected data paths.
- Started retiring copied learning-site surfaces through narrow verified waves.

### Quality

- Documented the Activity -> Assignment -> Attempt -> Results architecture.
- Kept public student payloads sanitized so answers remain server-side until
  scoring and allowed review.
- Added acceptance coverage for public pages, protected pages, activity
  authoring, publishing, and student attempts.
